🛕 Temple Fund Billing System

📌 Overview
The Temple Fund Billing System is a desktop application developed using PyQt5 to digitize and automate the process of managing temple fund receipts. The system replaces traditional paper-based receipts with a digital solution, improving efficiency, accuracy, and record-keeping.

🚀 Features
  ✅ Automated receipt generation
  ✅ Digital storage of all transactions
  ✅ Excel-based data management
  ✅ User-friendly GUI using PyQt5
  ✅ Analysis dashboard for fund tracking
  ✅ Easy record retrieval and transparency
  
🛠️ Technologies Used
  Python – Core logic implementation
  PyQt5 – Graphical User Interface
  Pandas / OpenPyXL – Excel file handling
  
⚙️ How It Works
  User enters donor and payment details through the UI
  System generates a digital receipt instantly
  Transaction is stored in an Excel sheet
  Data can be viewed and analyzed in the dashboard
  
📊 Analysis Module
  The system includes an analysis page that helps visualize:
  Total funds collected
  Daily/Monthly trends
  Transaction summaries
  
💡 Advantages
  Reduces manual errors
  Saves time and paperwork
  Ensures organized and secure records
  Easy auditing and reporting
  
🔮 Future Enhancements

  Online payment support
